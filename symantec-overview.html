Symantec DLP Overview

Symantec Data Loss Prevention is an enterprise DLP solution that provides comprehensive data protection across endpoints, network, storage, and cloud. Originally developed by Vontu and acquired by Symantec in 2007, the product is now owned by Broadcom following their 2019 acquisition of Symantec's enterprise security business.

Current Ownership

Symantec DLP is now part of Broadcom's Enterprise Security portfolio. Support and licensing are handled through Broadcom. The product may also be referred to as "Broadcom DLP" in newer documentation.

Product Suite Components

Endpoint DLP

Agent-based protection for Windows, macOS, and Linux endpoints. Monitors USB, print, clipboard, email clients, and applications.

Learn More

Network DLP

Network Monitor (passive), Network Prevent for Email/Web (inline). Inspects SMTP, HTTP/HTTPS, FTP, and IM traffic.

Learn More

Network Discover

Scans file servers, databases, SharePoint, Exchange, and cloud storage to find sensitive data at rest.

Learn More

Cloud DLP

Integration with CloudSOC (CASB) for SaaS application protection. API-based and inline scanning.

Learn More

Deployment Options

Traditional On-Premises Deployment

Full control over infrastructure with Enforce Server hosted in your data center.

Components Required:

  • Enforce Server — Central management platform (Windows Server + Oracle Database)
  • Detection Servers — Network Monitor, Network Prevent, Network Discover
  • Endpoint Agents — DLP Agent deployed to Windows/macOS/Linux
  • Oracle Database — Backend database for policies, incidents, and configuration
Infrastructure Consideration

On-premises deployments require Oracle Database licensing, which adds significant cost. Plan for database maintenance, backups, and high availability.

ICDm Cloud Console

Integrated Cyber Defense Manager (ICDm) provides cloud-based management for Symantec DLP.

Benefits:

  • No on-premises Enforce Server required
  • Cloud-hosted policy management and reporting
  • Automatic updates and maintenance
  • Integration with other Broadcom security products
  • Reduced infrastructure footprint

Limitations:

  • Some advanced features may not be available
  • Requires internet connectivity for management
  • Data processing in Broadcom cloud

Hybrid Deployment

Combine on-premises detection with cloud management, or mix on-prem and cloud-based scanning.

Common Hybrid Scenarios:

  • On-premises Network DLP + Cloud-managed Endpoint DLP
  • On-premises Enforce + CloudSOC for SaaS apps
  • Gradual migration from on-prem to ICDm

Detection Methods Overview

Method Acronym Description Use Case
Described Content Matching DCM Regex patterns, keywords, data identifiers Credit cards, SSNs, structured data
Indexed Document Matching IDM Document fingerprinting from source files Templates, contracts, source code
Exact Data Matching EDM Database record matching Customer lists, employee data
Vector Machine Learning VML ML-based content classification Unstructured documents, legal, financial

For detailed information on each detection method, see Detection Methods.

Licensing Model

Symantec DLP is licensed based on:

  • User Count — Per-user licensing for endpoint and network protection
  • Server Count — Additional licenses for detection servers
  • Module Selection — Endpoint, Network, and Discover are separate modules
  • Support Tier — Standard vs. Premium support options
Licensing Tip

Contact Broadcom or your reseller for current pricing. Many organizations are evaluating migration to Microsoft Purview or other cloud-native solutions due to licensing costs and infrastructure overhead.
