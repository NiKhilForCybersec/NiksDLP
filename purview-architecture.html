Microsoft Purview Architecture

Microsoft Purview DLP is built on a unified cloud architecture that integrates with Microsoft 365 services, Microsoft Defender for Endpoint, and Microsoft Defender for Cloud Apps.

Key Components

The central management portal at compliance.microsoft.com where administrators create policies, configure sensitive information types, and monitor DLP activity.

A single policy engine that evaluates content across all protected locations. Write once, protect everywhere.

Provides the endpoint sensor for Endpoint DLP. No additional agent deployment required if already using MDE.
