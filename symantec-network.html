Symantec Network DLP

Symantec Network DLP provides traffic inspection at network boundaries to detect and prevent sensitive data exfiltration via email, web, and other network protocols.

Network Monitor

Network Monitor provides passive traffic inspection via network TAP or SPAN port. It detects but does not block sensitive data.

Network Prevent for Email

Network Prevent for Email integrates with your mail transfer agent (MTA) to block, redirect, or modify emails containing sensitive data.

Integration Options

Reflection Mode (Recommended)

MTA sends copy of email to Network Prevent; waits for verdict before delivery.

  • MTA remains authoritative for email routing
  • DLP server returns allow/block/modify decision
  • Supports most MTAs (Exchange, Postfix, Sendmail)
  • Timeout handling prevents mail delays

MTA Integration Mode

Network Prevent acts as hop in mail flow.

  • All email routed through DLP server
  • Direct inspection and modification
  • Requires careful HA planning
  • Single point of failure risk

Forwarding Mode

BCC/journal copy sent to DLP for monitoring only.

  • No blocking capability
  • Detection and alerting only
  • Lowest impact deployment
  • Good for initial assessment

Response Actions

  • Block — Reject email, send NDR to sender
  • Redirect — Send to quarantine or compliance review
  • Modify — Remove attachment, add disclaimer
  • Encrypt — Apply email encryption
  • Tag — Add header for downstream processing

Network Discover scans data repositories to find sensitive data at rest.

Lotus NotesNotes APIScan Notes databases
Box, O365REST APICloud storage scanning

Scan Scheduling

Best Practice Schedule 
# Initial full scan
- Schedule during off-hours (weekends)
- Expect 1-5 GB/hour depending on content
- May take days for large repositories

# Incremental scans
- Daily or weekly after initial scan
- Only scan new/modified files
- Much faster than full scan

# Targeted scans
- High-risk locations (HR shares, Finance)
- More frequent (daily)
- Smaller scope, faster completion
