DLP Troubleshooting Overview

This section provides diagnostic approaches for common DLP issues across platforms.

General Troubleshooting Approach

  1. Define the Problem

    What's the expected behavior vs. actual behavior? When did it start?

  2. Check Recent Changes

    Policy updates, agent deployments, infrastructure changes?

  3. Verify Agent Health

    Is the agent installed, running, and communicating?

  4. Review Logs

    Check application logs, event logs, and service status.

  5. Test in Isolation

    Can you reproduce with a simple test case?
